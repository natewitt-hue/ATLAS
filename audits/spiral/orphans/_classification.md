2. **One-off utilities** (`backfill_embeddings.py`, `upload_emoji.py`, `espn_asset_scraper.py`) — manual-run admin scripts. Worth either documenting in README or moving to `scripts/`.
3. **Truly dead helpers** (`embed_helpers.py`) — likely refactored away when atlas_send took over.
